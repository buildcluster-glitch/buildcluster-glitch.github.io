# ビルドクラスタ 業務アプリ ポータル

(株)ビルドクラスタの業務用Webアプリ集です。すべてブラウザだけで使えます（インストール不要）。スマホ・PCどちらでも動きます。

🌐 **入口URL（共通）**
https://buildcluster-glitch.github.io/

---

## 📋 アプリ一覧

ポータルを開くと、利用可能なアプリのカードが並びます。

| アイコン | アプリ名 | 用途 | 詳細マニュアル |
|---|---|---|---|
| 📋 | **タスクマネージャー** | 担当者別タスク・優先度・期日をチームで共有 | 別ページ参照 |
| 🔧 | **原状回復・修繕工事進捗管理** | 物件ごとの工事進捗と発注書を現場で管理 | 別ページ参照 |
| 📦 | **在庫管理** | 消費・補充・棚卸・発注 | 別ページ参照 |
| 📖 | **業務アプリ 説明書** | 詳細マニュアル | https://buildcluster-glitch.github.io/manual/ |

> 各アプリの詳しい使い方は、それぞれの個別マニュアルを参照してください。

## 🔐 初回アクセス（パスワード）

セキュリティ確保のため、初回アクセス時にパスワード入力画面 🔒 が表示されます。

1. パスワードを入力 → 「入る」
2. 一度入力すると、その端末で **30日間** はパスワード不要
3. 30日後 or 別の端末では再度パスワード入力

> **パスワードは社内で共有**。外部に漏らさないでください。
> 変更が必要な場合は管理者まで。

## 🚀 各端末で1回だけやる初期設定

### 1. 入口URLをブックマーク or ホーム画面に追加

**iPhone (Safari)**
1. https://buildcluster-glitch.github.io/ を Safari で開く
2. 画面下の共有ボタン `[↑]` をタップ
3. 「**ホーム画面に追加**」を選択

**Android (Chrome)**
1. https://buildcluster-glitch.github.io/ を Chrome で開く
2. 右上の `︙` メニュー
3. 「**ホーム画面に追加**」を選択

**PC (Chrome / Edge)**
1. アドレスバー右の `⋮` または `…` メニュー
2. 「ブックマークに追加」または「**アプリとしてインストール**」

ホーム画面に追加すると、ブラウザ画面を経由せず、アプリのように直接起動できます。

### 2. パスワード入力 → 中に入る

最初の起動時にだけパスワード入力画面が出ます。
30日間は次回以降スキップされます。

### 3. 自分の名前を設定（アプリごと）

各アプリのヘッダーにある「**未設定**」または「👤 アイコン」をタップし、自分の名前を選択してください。
これにより、誰が操作したか自動で記録されるようになります。

### 4. GAS Web App URL / トークンを設定（アプリごと）

「原状回復・修繕工事進捗管理」「在庫管理」には、クラウド同期用の **GAS URL** と **トークン** の入力が必要です。

- 入力場所: 各アプリの設定または ⬇ ボタン
- 値: 社内共有の管理者から受け取った値を貼り付け
- 一度入れれば次回以降は記憶されます

「タスクマネージャー」は外部 Google Apps Script で動作するアプリなので、URLを開けばそのまま使えます（個別設定不要）。

詳細手順は各アプリの個別マニュアルを参照してください。

## 💡 動作環境

| 項目 | 推奨 | 備考 |
|---|---|---|
| スマホ | iPhone (iOS 14+) / Android (Chrome 90+) | カメラ機能含めて主要機能対応 |
| PC | Chrome / Edge (最新版) | Safari/Firefox も基本動作 |
| 通信 | インターネット必須 | 初回起動とクラウド同期で必要 |
| データ保存 | ブラウザの localStorage | 端末ごとに保存・GAS経由でクラウド同期 |

## 🆘 困った時

**ロック画面が消えない / パスワードを忘れた**
→ 管理者にパスワードを確認

**パスワードを入れたのに再度求められる**
→ ブラウザのキャッシュ・履歴を消去すると localStorage がクリアされて再入力が必要になります。30日カウントもリセットされます。

**アプリが古いまま更新されない**
→ ブラウザで強制リロード（`Ctrl + Shift + R` / `Cmd + Shift + R`）
→ それでも直らない場合は F12 → Application → Clear storage

**自分の操作が他の人の端末で見られない**
→ 各アプリの ☁ 同期ボタンを押してください（または1分待つ）
→ それでも反映されない場合は管理者まで
